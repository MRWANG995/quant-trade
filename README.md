## вңЁ ж ёеҝғзү№жҖ§

### рҹ“Ҡ зңҹе®һиЎҢжғ…пјҢйӣ¶йўқеӨ–й…ҚзҪ®
- **9 дёӘе“Ғз§Қ**пјҡBTC + XAUUSD + Magnificent 7пјҲAAPL / MSFT / GOOGL / AMZN / NVDA / META / TSLAпјү
- **еӨҡжәҗи°ғеәҰ**пјҡ
  - BTC вҶ’ **Coinbase Exchange**пјҲе…¬ејҖгҖҒе…Қ keyгҖҒе…ЁзҗғеҸҜз”ЁпјӣBinance зҫҺеӣҪ 451 ж—¶еӨҮз”Ёпјү
  - XAU + зҫҺиӮЎ вҶ’ **Stooq**пјҲдёҖдёӘе…Қиҙ№ key жӢүж»Ў ~3 е№ҙж—Ҙ Kпјү
  - Alpha Vantage / Yahoo Finance дҪң fallback

### рҹ§  17+ еҶ…зҪ®зӯ–з•Ҙ + дёүз§Қжү©еұ•жЁЎејҸ
**з»Ҹе…ёжҠҖжңҜзӯ–з•Ҙ**
- еҸҢеқҮзәҝдәӨеҸү В· RSI и¶…д№°и¶…еҚ– В· MACD В· еёғжһ—еёҰеқҮеҖјеӣһеҪ’
- е”җеҘҮе®үйҖҡйҒ“пјҲжө·йҫҹпјү В· MACD и¶ӢеҠҝиҝҮж»Ө В· еҠЁйҮҸи¶ӢеҠҝ

**LLM еҜ№иҜқз”ҹжҲҗзӯ–з•ҘпјҲDSLпјү** вҖ” з”ЁдёҖеҸҘдёӯж–ҮжҸҸиҝ°пјҢGemini 2.5 зӣҙжҺҘз”ҹжҲҗеҸҜеӣһжөӢзҡ„ JSON DSLпјҡ
> *"RSI и·Ңз ҙ 30 еҗҺеӣһеҚҮеҒҡеӨҡпјҢиҝӣе…Ҙ 70 и¶…д№°еҮәеңә"*
> вҶ’ `{op: crosses_above, indicator: rsi(14), const: 30}` ...

**з»„еҗҲзӯ–з•ҘпјҲCompositeпјү** вҖ” еӨҡдёӘеӯҗзӯ–з•ҘеҠ жқғиҒҡеҗҲпјҡ
- еҠ жқғиҒҡеҗҲпјҡжҜҸдёӘеӯҗзӯ–з•ҘдёҖдёӘжқғйҮҚпјҢиҫ“еҮәеҠ жқғеҮҖеҲҶеӨ§дәҺ 0 еҒҡеӨҡгҖҒе°ҸдәҺ 0 еҒҡз©ә
- дҫӢпјҡMA 50% + RSI 30% + Donchian 20%

**Agent зңӢзӣҳејҸзӯ–з•Ҙ** вҖ” Gemini е®һж—¶еҲҶжһҗеҚ•дёӘж Үзҡ„пјҢиҫ“еҮә**еҸҜжү§иЎҢзҡ„дәӨжҳ“и®ЎеҲ’**пјҡ
- зі»з»ҹжҸҗзӨәиҜҚжҺ§еҲ¶ Agent йЈҺж јпјҲ"дҪ жҳҜи¶ӢеҠҝеһӢдәӨжҳ“е‘ҳпјҢе…іжіЁ close жҳҜеҗҰз«ҷдёҠ SMA50вҖҰ"пјү
- Gemini зңӢжңҖиҝ‘ 60 ж № K зәҝ + RSI/SMA/EMA/ATR + йҖҡйҒ“жҢҮж Ү вҶ’ иҫ“еҮә `{side, confidence, entry_price, stop_loss, take_profit, reason}`
- д»·дҪҚзәҝзӣҙжҺҘз”»еңЁ K зәҝеӣҫдёҠпјҡрҹҹў е…Ҙеңә / рҹ”ҙ жӯўжҚҹ / рҹҹЎ жӯўзӣҲ
- жҢү (зӯ–з•Ҙ, ж Үзҡ„, иЎҢжғ…ж—Ҙжңҹ) зј“еӯҳпјҢйҮҚеӨҚжү“ејҖеҗҢдёҖе“Ғз§ҚдёҚзғ§ tokenпјӣ"йҮҚз®—"жҢүй’®ејәеҲ¶еҲ·ж–°
- еҗҢж ·еҸҜз”ЁдәҺжҜҸж—Ҙжү№йҮҸжү«жҸҸпјҢжҢү max_trades_per_day йҖү top-K еҮәеҚ•

### рҹ“Ҳ е®Ңж•ҙзҡ„еӣһжөӢеј•ж“Һ
- Equity / Drawdown / жңҲж”¶зӣҠзғӯеҠӣеӣҫ / ж Үзҡ„жӢҶи§Ј
- жҢҮж ҮпјҡSharpe / Sortino / Calmar / жңҖеӨ§еӣһж’Ө / жңҖй•ҝж°ҙдёӢеӨ©ж•° / иғңзҺҮ / зӣҲдәҸжҜ” / е№іеқҮжҢҒд»“еӨ©ж•°
- ж— йЈҺйҷ©еҲ©зҺҮиҮӘеҠЁд»Һ FRED жӢүеҸ–пјҢиҗҪеә“

### рҹҺҜ жЁЎжӢҹзӣҳдәӨжҳ“
- жҜҸж—Ҙжү«жҸҸ вҶ’ йҖү Top-K дҝЎеҸ· вҶ’ йЈҺжҺ§пјҲжҜҸж—ҘдёҠйҷҗгҖҒеҚ•ж Үзҡ„йҷҗйў‘пјүвҶ’ дёӢеҚ•
- жЁЎжӢҹзӣҳжҢүдёӢдёҖдәӨжҳ“ж—ҘејҖзӣҳд»·жҲҗдәӨпјҢеҗ«еҸҜй…ҚзҪ®ж»‘зӮ№е’ҢжүӢз»ӯиҙ№
- IB / OANDA йҖӮй…ҚеҷЁйӘЁжһ¶е·Іе°ұз»ӘпјҢеҫ…иЎҘе…ЁеҸҜдёҠе®һзӣҳ

### рҹ”Қ зӯ–з•Ҙзә§иҝҮж»Ө
- еӣһжөӢеҺҶеҸІ / е§”жүҳи®°еҪ• / жү«жҸҸж—Ҙеҝ— дёүдёӘйЎөйқўйЎ¶йғЁйғҪжңү"жҢүзӯ–з•ҘиҝҮж»Ө"дёӢжӢү
- URL жҢҒд№…еҢ–пјҲ`?strategy_id=X`пјүпјҢж–№дҫҝеҜ№жҜ”еҗҢдёҖзӯ–з•ҘеӨҡж¬Ўи·‘зҡ„з»“жһң

## рҹҡҖ еҝ«йҖҹејҖе§Ӣ

### 1. жӢүд»Јз Ғ + еӨҚеҲ¶й…ҚзҪ®

```bash
git clone https://github.com/MRWANG995/quant-trade.git
cd quant-trade
cp backend/.env.example backend/.env
```

### 2. жӢҝдёҖдёӘе…Қиҙ№ Stooq API KeyпјҲ30 з§’пјү

жү“ејҖ https://stooq.com/q/d/?s=eurusd&get_apikey вҶ’ иҫ“йӘҢиҜҒз Ғ вҶ’ д»Һеә•йғЁдёӢиҪҪй“ҫжҺҘзҡ„ URL йҮҢеӨҚеҲ¶ `apikey=` еҗҺйӮЈдёІгҖӮ

еЎ«иҝӣ `backend/.env`пјҡ
```
STOOQ_API_KEY=<дҪ зҡ„ key>
```

### 3. пјҲеҸҜйҖүпјүжӢҝдёҖдёӘ Gemini API Key

- з”Ё LLM з”ҹжҲҗ DSL зӯ–з•ҘжҲ– Agent LLM зӯ–з•Ҙж—¶йңҖиҰҒ
- е…Қиҙ№жЎЈ 1500 ж¬Ў/еӨ©пјҢи¶іеӨҹдёӘдәәдҪҝз”Ё
- https://aistudio.google.com/apikey вҶ’ Create API key
- еЎ«иҝӣ `backend/.env`пјҡ`GEMINI_API_KEY=<key>`

### 4. еҗҜеҠЁеҗҺз«Ҝ

```bash
cd backend

PYTHONPATH=. alembic upgrade head
PYTHONPATH=. uvicorn app.main:app --reload --port 9999
```

### 5. еҗҜеҠЁеүҚз«Ҝ

```bash
cd frontend

```

жү“ејҖ http://localhost:9998

### зӯ–з•Ҙзұ»еһӢжү©еұ•жңәеҲ¶

жүҖжңүзӯ–з•Ҙиө°з»ҹдёҖ `STRATEGY_TYPE_CATALOG`пјҢж–°еўһзӯ–з•Ҙзұ»еһӢеҸӘйңҖпјҡ
1. `app/strategies/<type>.py` е®һзҺ° `scan_historical_signals` / `latest_signal` / `min_bars_required` дёүдёӘеҮҪж•°
2. `registry.py` еңЁ `STRATEGY_TYPE_CATALOG` еҠ  label + param_schema
3. `_HANDLERS` жіЁеҶҢдёүдёӘеҮҪж•°
4. пјҲеҸҜйҖүпјүеңЁ `validate_params` йҮҢеҠ иҮӘе®ҡд№үж ЎйӘҢ

еӣһжөӢеј•ж“Һе’ҢжҜҸж—Ҙжү«жҸҸиҮӘеҠЁж”ҜжҢҒж–°зұ»еһӢгҖӮ
